# Bachelor’s Thesis – React Application with Vite - AUEB Career Site

This project is a frontend web application developed as part of a bachelor’s thesis.  
The application is built using **React** and **Vite**, focusing on modern web development practices and **UX/UI design principles**, with emphasis on usability and user experience.

---

## Technologies Used

- React
- Vite
- JavaScript
- HTML5 / CSS3
- npm

---

## Prerequisites

To run this project locally, the following are required:

- Node.js (version 18 or later)
- npm

Check installation:

```bash
node -v
npm -v
```
